Sääʹmteʹǧǧ jueʹjji stipeeʹndid nellja Sääʹmvuuʹd škoouʹlʼjemkõõskõõzzâst valmštõõvvi reʹttev sääʹmǩiõl mättʹtõõttja. Saamelaiskäräjät antoi stipendin neljälle Saamelaisalueen koulutuskeskuksesta valmistuneelle ahkeralle saamen kielen opiskelijalle.
Sääʹmvuuʹd škoouʹlʼjemkõõskõõzz ǩiđđprääʹzneǩ prääzkješ seärad ǩieʹssmannu kuälmad peeiʹv. Saamelaisalueen koulutuskeskuksen kevätjuhlaa vietettiin keskiviikkona kesäkuun kolmantena päivänä.

Aanarsääʹm: Jari Linjama Inarinsaame: Jari Linjama
