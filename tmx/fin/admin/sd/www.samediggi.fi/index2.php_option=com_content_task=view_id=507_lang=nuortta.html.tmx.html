ÄäŠšš ǩiõttŠtõõlât 24.8.2011 tâŠvvjânnmlaž meertiõrvâsvuõttkonfereenssâst Turkust. Asiaa käsitellään 24.8.2011 pohjoismaisessa kansanterveyskonferenssissa Turussa.
